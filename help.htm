Online Help:

The Public Comment application allows citizens to submit requests for service in their community. It is a single application that can be used on a desktop browser, mobile phone, and tablet device. This application provides 24x7 access to the organization and typically supplements customer service phone numbers staffed by local governments.

Using the Application

The Public Comment application allows citizens to submit requests for service. Search for an address or click/tap directly on the map to locate your request for service. Then, provide details about the request and click/tap Submit to initiate your request. You can also attach a photo, video or document to your request. After you click/tap Submit, you'll be provided a Request # that can be used to track the status of your request. If you find a request has already been submitted by someone else at the location you're interested in, you can select the existing service request. Then provide additional comments and rank the importance of this issue to you. In local governments that have deployed an automated system (CRM/ CMMS systems) to track service requests and work activities, the online requests will be routed to staff responsible for its resolution. If an automated is not present in the local government, a manual service ticket will typically be created and routed to appropriate staff.

Finding an Area of Interest

There are two primary ways to identify an area of interest. You can interact directly with the map and click/tap on a service request. You can also search for an address.

Map Based Navigation (On a Desktop PC)

There are several ways to interact with the map:

1. Use the basemap switcher to select the basemap you'd like to interact with
2. Hold down the shift key and drag a box on the map to zoom in
3. Hold Down Ctrl+Shift to zoom out
4. Roll your mouse roller to zoom in/out
5. Click the zoom in/out button

Map Based Navigation (On a Touch Screen Device)

There are two ways to interact with the map:

1. Pinch to zoom in/out
2. Tap the zoom in/out button

Searching

The search box can be used to find an address you're interested in:
1. Enter the address you're interested in
3. Click on the item in the list to locate the address on the map

You can enter a partial address and then identify the correct value in the list.

Service Requests

Search for an address or click directly on the map to locate your request for service. Then, provide details about the request and click Submit to initiate your request. You'll be provided a Request # that can be used to track the status of your request. If you find a request has already been submitted by another party, you can click on the existing request, provide additional comments and increase the importance of the request.

More Information

The Public Comment application can be used by water utilities, public works agencies, or other local government organizations to deliver a single web-based service request application that can be accessed on multiple computing devices. This application provides 24x7 access to the organization and typically supplements customer service phone numbers staffed by local governments.

Please visit Esri's ArcGIS for Local Government Resource Center for more information.

Browser Compatibility

The Public Comment application is supported on the following browsers: Microsoft IE 7/8/9, Mozilla Firefox, Google Chrome and Apple Safari.
